A work-in-progress utility to decode hex-encoded Windows Hardware Event Architecture (WHEA) records.

- [Requirements](#requirements)

Requirements
------------

- .NET Framework 4.6.2 (or newer)  
  *Built-in since Windows 10 1607 and Server 2016.*
